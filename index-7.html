The Protocol

Two things, clearly separated.

A live, tested blockchain — and a token that has not launched yet. We keep that distinction visible everywhere on this site.

Live · v0.1.0-mvp

SAYANJALI BLOCKCHAIN

An independent Layer-1 network built from scratch in Python — its own consensus, its own block format, no fork of an existing chain. It ships with Proof-of-Work consensus, ECDSA wallets, signed transactions, a REST API, and a CLI.

Proof-of-Work ECDSA Wallets REST API CLI 37/37 tests
Planned · Pre-launch

SYJ Token

SYJ is designed to settle natively on SAYANJALI BLOCKCHAIN. It's intended for network fees, governance participation, and access to ecosystem services — AI tools, developer infrastructure, and compliance software — not as a speculative instrument.
