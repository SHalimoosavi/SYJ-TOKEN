FAQ
Common Questions
What is a Layer 1 blockchain?+

A Layer 1 blockchain is a base, independent network with its own consensus mechanism, block format, and native token — as opposed to a token deployed on top of an existing chain like Ethereum. SAYANJALI BLOCKCHAIN is a Layer 1 network built from scratch in Python.

What is SAYANJALI BLOCKCHAIN?+

SAYANJALI BLOCKCHAIN is an independent Layer-1 network with Proof-of-Work consensus, ECDSA wallets, signed transactions, a REST API, and a CLI. It shipped as v0.1.0-mvp with 37 passing automated tests and is publicly available on GitHub.

What is the SYJ Token?+

SYJ is a planned utility token designed to settle natively on SAYANJALI BLOCKCHAIN. It is intended for network fees, governance participation, and access to ecosystem services such as AI tools, developer infrastructure, and compliance software — not as a speculative investment.

Has the SYJ Token launched yet?+

No. The SYJ Token has not launched. The Token Generation Event (TGE) is planned and will be announced through official channels. The underlying blockchain, SAYANJALI BLOCKCHAIN, is already live as an open-source MVP.

Will smart contracts be supported?+

Smart contract execution is on the public roadmap, planned after peer-to-peer networking and testnet milestones. It is not yet implemented in the current v0.1.0-mvp release.

Is SAYANJALI BLOCKCHAIN open source?+

Yes. The full source code is publicly available on GitHub, including its consensus engine, wallet implementation, REST API, and test suite.

Who is building SAYANJALI BLOCKCHAIN?+

SAYANJALI BLOCKCHAIN is built by Syed Ali Hasan Moosavi, Founder and Managing Director of SAYANJALI NEXUS PRIVATE LIMITED, a Hyderabad-based technology company.

How can developers contribute?+

Developers can explore, fork, and contribute to the SAYANJALI BLOCKCHAIN repository on GitHub, or reach out via the contact section below to discuss collaboration.
